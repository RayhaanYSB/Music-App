A social music tracking platform where users can track, rate, and share their favorite albums and songs - like Letterboxd but for music.

## Features (Planned)
- Track albums and songs you've listened to
- Rate and review music
- Create lists and collections
- Follow friends and see their activity
- Discover new music through social recommendations

## Tech Stack
- **Frontend:** React
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Authentication:** JWT
